Maven installation in Windows

How to install Maven in Windows?

How to install Maven?

 

First download Maven zip file, extract it and configure the Windows environment path variable.

 

Make sure you have installed JDK and JAVA_HOME variable points to JDK folder. 

 

Add new variable MAVEN_HOME and point it to your Maven folder.

 

Update Path variable, append: ;%MAVEN_HOME%\bin

 

Now open command line and type mvn -version to test installation.
